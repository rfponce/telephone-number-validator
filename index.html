Telephone Number Validator

This program validates if a number has a valid US telephone number format. If the number is invalid the text in the textbox will become red. Otherwise will become green. The following formats are accepted:

  • 555-555-5555
  • (555)555-5555
  • (555) 555-5555
  • 555 555 5555
  • 5555555555
  • 1 555 555 5555
